## ❓ FAQ

**Q: How do I change the colors?**
A: Edit `:root` in `src/App.css`. See CUSTOMIZATION_GUIDE.md.

**Q: Can I add more sections?**
A: Yes! Create a new component in `src/components/` and add to App.jsx.

**Q: Does it work on mobile?**
A: Yes! Fully responsive with hamburger menu on mobile.

**Q: How do I deploy it?**
A: See DEPLOYMENT.md for 4 easy options (GitHub Pages, Vercel, Netlify, AWS).

**Q: Can I use my own domain?**
A: Yes! See "Custom Domain" section in DEPLOYMENT.md.

**Q: Is there a contact form backend?**
A: Form UI included. Add EmailJS or SendGrid for email functionality.

**Q: Can I add a blog?**
A: Yes! It's listed as optional enhancement in PROJECT_SUMMARY.md.

## 🐛 Troubleshooting

### Port already in use
```bash
npm run dev -- --port 3000
```

### Changes not showing
- Press F5 to refresh
- Close and reopen server: Ctrl+C then `npm run dev`

### Build fails
```bash
rm -r node_modules
npm install
npm run build
```

### Styling looks wrong
- Clear browser cache: Ctrl+Shift+Delete
- Test in incognito/private mode

See QUICKSTART.md for more help.
